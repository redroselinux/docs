# Redrose Linux Documentation

Hello! This site is the wiki for Redrose Linux. Learn how to install, contribute to, or use the new independent Linux distribution. Use the sidebar on the left to navigate the site.

> **Important**<br>
> Redrose Linux is in alpha. It is not AT ALL ready for almost any usage. We continue working on Redrose and hopefully we can get a beta release at the end of 2026.

**If you got to this page and wish to view our main page, visit [redroselinux.org](https://redroselinux.org).**
